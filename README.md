# シノビ漂流記

## 概要
イカダに乗った忍者が、迫りくる障害物を破壊し、敵の攻撃を避けながらゴールの島を目指す2D横スクロールシューティングゲームです。
「イカダが壊れる前に、そして攻撃に当たらないようにゴールできるか」という緊張感のある要素が特徴です。

## 開発体制
サークル内の4日間ゲーム制作イベントにて、12名のチームで開発（ポートフォリオ掲載許可済）。
- **開発期間:** 4日間
- **プラットフォーム:** PC (Windows)
- **開発環境:** Unity 2022.3.20f1 / C# / Visual Studio Code

## 担当箇所
12名のチームの中で、主に**「ゲームコアシステムの一部」と「データ連携・リザルト」**を担当しました。

**担当したソースコード一覧:** [Assets/Scripts/1Haruso](./Assets/Scripts/1Haruso)

### 1. ゲームオブジェクト・ギミック
- **漂流物のシステム:** 流れてくる漂流物の生成や移動ロジックと、イカダ接触時に破壊するシステムの実装。
- **判定処理:** プレイヤーが海に落ちた落下判定（ゲームオーバー）の実装。

### 2. シーン間連携とリザルト
- **データ管理:** プレイ中の取得したバフ・デバフアイテム数を保持し、シーンを跨いでリザルト画面に引き継ぐ機能を実装。
- **UI表示:** スコアに応じたリザルト画面のアニメーション・数値表示。

### 3. エンドレスモード (個人担当)
通常モードとは別に、長く遊べるエンドレスモードをゼロから実装しました。
- ランダム生成アルゴリズムによる障害物・アイテムの配置。
- リアルタイムでの走行距離計測システム。
